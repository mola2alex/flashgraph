Constructor detail
Environment()constructor
public function Environment(... args)

Creates a new instance of an Environment. This establishes a large number of default functions, constants and operators.

Parameters
... args — The constructor may be given no arguments, an array of Strings, or a list of Strings. In the first case, no independent variable names are set up. In the latter cases, the given strings are passed along to the addVars method.

Throws
— ArgumentError: A variable name is invalid or conflicts with another identifier.

Example
The following code results in three equivalent Environments. 
    var env1:Environment = new Environment()
    env1.addVars(["x", "y"]);
    var env2:Environment = new Environment(["x", "y"]);
    var env3:Environment = new Environment("x", "y");

Method detail
addAddOp()method
public function addAddOp(opName:String, fn:Function):void

Add an operator with the same precedence as addition to the given instance.

Parameters
opName:String — The name of the operator. Little sanity checking is done on this argument, but it must be a one-character symbol and not a known identifier (i.e., isKnownId must return false). If either of these is not the case, an ArgumentError will be thrown.
 
fn:Function — The ActionScript function corresponding to the operator name. If it is not binary (i.e., if it does not take exactly 2 arguments), an ArgumentError will be thrown. Its arity will be determined from its length property, so functions that use the ... notation for accepting a variable number of arguments may not work as intended.

Throws
— ArgumentError: The operator name conflicts with another identifier or fn is not binary.

See also

isKnownId()

isValidId()method 
public function isValidId(id:*):Boolean

Returns true if the given string is a valid name for an identifier in this Environment. Note that this does not check for naming conflicts; use isKnownId for that purpose.

Parameters
id:* — The identifier to test. A valid identifier begins with a letter or an underscore and then contains some number of letters, numbers, or underscores.
