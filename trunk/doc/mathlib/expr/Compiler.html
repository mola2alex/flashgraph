compile()method
public static function compile(env:Environment, s:String, optimize:Boolean = true):CompiledFn

Compile a string into an evaluable CompiledFn.

Parameters
env:Environment — The Environment to use for lookups of functions, operators, constants, and variable names.
 
s:String — The expression string to tokenize, parse, and compile.
 
optimize:Boolean (default = true) — If true, the compiler will attempt to optimize the resulting CompiledFn for speed. At the moment, this just collapses constant subtrees of the ParseTree to a single number.

Returns
CompiledFn

public static function compileParseTree(env:Environment, pt:ParseTree, optimize:Boolean = true):CompiledFn

Compile a ParseTree (generated by the Parser) into an evaluable CompiledFn.

Parameters
env:Environment — The Environment to use for lookups of functions, operators, constants, and variable names.
 
pt:ParseTree — The ParseTree representing the parsed expression to compile. This may be generated by Parser.parse().
 
optimize:Boolean (default = true) — If true, attempt to optimize the resulting CompiledFn. See the compile method for more details.
