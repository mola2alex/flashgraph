Constructor detail
FnGraph()constructor
public function FnGraph(n_parentGrapher:Grapher2D, n_fn:Function, n_lineStyle:LineStyle = null)

Creates a new FnGraph object. End-users should never have reason to call this; instead, use the addFnGraph() method of a Grapher2D instance.

Parameters
n_parentGrapher:Grapher2D — The Grapher2D instance with which this graph is associated.
 
n_fn:Function — The function to graph. This function must be capable of being called with one Number argument and must return a Number.
 
n_lineStyle:LineStyle (default = null) — The line style to use when drawing the graph of the function. If it is given as null, LineStyle.Hairline is used.
