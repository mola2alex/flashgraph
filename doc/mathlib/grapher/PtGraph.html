PtGraph()constructor
public function PtGraph(n_parentGrapher:Grapher2D, n_pts:Array, n_connected:Boolean = true, n_showPoints:Boolean = true, n_lineStyle:LineStyle = null)

Create a new PtGraph object. There should never be a reason to call this; instead, use the addPtGraph() method of an instance of Grapher2D.

Parameters
n_parentGrapher:Grapher2D — The Grapher2D instance with which this graph is associated.
 
n_pts:Array — An array of flash.geom.Points to be drawn.
 
n_connected:Boolean (default = true) — True to draw lines connecting the points in order of their appearance in _pts
 
n_showPoints:Boolean (default = true) — True to draw a small circle at the location of each point. Note: if connected and this property are both false, there will be nothing to draw.
 
n_lineStyle:LineStyle (default = null) — The style of line to use for drawing connecting lines if connected is true and for points if showPoints is true. If its value is null, LineStyle.Hairline is used.

See also
