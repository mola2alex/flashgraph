Classpublic class Lexer

Generates an array of Tokens from a string expression. Note that this may not be a one-to-one representation of the incoming string; whitespace is insignificant and implicit multiplication operators are inserted where appropriate if the doImplicitMult flag in the Environment is true. For example, the lexer would make the following conversions in an Environment with independent variables x and y: 
  xy => x * y
  xsin(y) => x * sin(y)
  sin(x)exp(y) => sin(x) * exp(y)
If doImplicitMult were false, the above examples would generate errors.
